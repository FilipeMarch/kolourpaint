How to make a checkinstall package of KolourPaint.
--------------------------------------------------

change hostname to localhost.localdomain

build using apbuild FIRST

become root

make doc-pak/
copy docs except INSTALL into it
make it world readable

cp README description-pak
edit to just have description
remove opensource ref in description

checkinstall -R --docdir=/usr/share/doc --inspect --review-spec --backup=no --fstrans=no

remove /usr/share/doc/HTML/en/common from list
