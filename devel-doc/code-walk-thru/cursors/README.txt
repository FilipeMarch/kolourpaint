
cursors/ Package


Provides KolourPaint-specific mouse cursors.


[>>>] kpCursorProvider

Factory for KolourPaint-specific mouse cursors (currently, only
kpCursorLightCross).

Used by kpToolFlowBase.


[>>>] kpCursorLightCross

A lighter version of Qt::CrossCursor.

Created by kpCursorProvider and used by kpToolFlowBase for tools with
square or diverse brushes (Brush, Eraser and Color Eraser).
